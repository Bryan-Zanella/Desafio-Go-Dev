Desafio Go Dev

Este programa foi feito com a linguagem Java utilizando o NetBeans IDE 8.2

Este projeto é um evento no qual as pessoas serão divididas em salas com lotação variável. 
Serão realizados também dois intervalos de café em espaços distintos.

Nele é possível você cadastrar as pessoas, colocando o nome e sobrenome.
Também você pode cadastrar as salas do evento, colocando o nome da sala e sua lotação, e, igualmente
com os espaçoes do café, colocando o local que será o café e sua lotação.

Para executar o programa siga este passo a passo:
Abra o projeto DesafioGoDev -> Pacote de Código-fonte -> desafiogodev.view -> TelaPrincipal.java
após, você clica com o botão direito do mouse em cima do "TelaPrincipal.java", e clique em Executar Arquivo.


Banco de Dados

Para alterar o user, password e localhost do sistema para conectar com o banco de dados, siga este passo a passo:
Abra o projeto DesafioGoDev -> Pacote de Código-fonte -> desafiogodev.dao -> Configuracao.java
após você desce ate onde está o "public Configuracao() {" e abaixo terá o user, password e localhost.


O código do banco de dados é:

create database desafiogodev;
use desafiogodev;

create table pessoas(
    codigo_pessoas integer not null auto_increment,
    nome varchar(50) not null,
    sobrenome varchar(50) not null,
    primary key (codigo_pessoas));

create table salas(
    codigo_salas integer not null auto_increment,
    nome varchar(50) not null,
    lotacao varchar(20) not null,
    primary key (codigo_salas));

create table espaco(
    codigo_espaco integer not null auto_increment,
    nome varchar(50) not null,
    lotacao varchar(20) not null,
    primary key (codigo_espaco));
